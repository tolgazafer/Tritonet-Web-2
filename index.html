What it does

🌍

Brainwave Mapping

Delta, Theta, Low/High Alpha, Low/High Beta, and Gamma bands each control dedicated MIDI CC values.

🎵

SATB Harmony

Real-time four-part chord voicing for Bass, Tenor, Alto, and Soprano with configurable chord types.

🎧

Web MIDI Output

Sends MIDI directly from your browser — no plugins. Works with any DAW or hardware synth.

👁

p5.js Visualization

Real-time canvas animation reflects the current harmonic state of the brainwave model.

🔄

OSC Support

Toggle OSC output alongside MIDI for integration with Max/MSP, SuperCollider, or Touch Designer.

💾

Single File

Everything — p5.js, fonts, logic — is bundled in one HTML file. No server, no install, no build step.

Get Tritonet

A single HTML file. Save it, open it in Chrome, allow MIDI access, and start playing.
